# Subham Karmakar - Portfolio

A modern, performant portfolio website built with Next.js 16, React 19, and TailwindCSS. Features smooth animations, dark/light theme support, and optimized for both SEO and LLM discoverability.

**Live Site:** [subham12r.dev](https://subham12r.dev)

## Tech Stack

- **Framework:** Next.js 16 (App Router, Turbopack)
- **Language:** TypeScript
- **Styling:** TailwindCSS 4
- **Animations:** GSAP, Framer Motion, Lenis (smooth scroll)
- **3D Graphics:** Three.js, React Three Fiber, Drei
- **UI Components:** Radix UI, Lucide Icons, Vaul (drawer)
- **Theming:** next-themes
- **Analytics:** Vercel Analytics

## Features

- Responsive design with mobile-first approach
- Dark/Light theme with system preference detection
- Smooth scroll animations with GSAP and Lenis
- GitHub contribution calendar integration
- Dynamic GitHub star count fetching
- Blog with markdown content
- Project showcase with live demos
- Contact form with modal
- SEO optimized with sitemap and robots.txt
- LLM optimized with llms.txt

## Project Structure

```
src/
├── app/                    # Next.js App Router pages
│   ├── blog/              # Blog pages
│   ├── projects/          # Projects page
│   ├── hackathons/        # Hackathons page
│   ├── photography/       # Photography page
│   └── page.tsx           # Home page
├── components/
│   ├── layout/            # Navigation, Footer, etc.
│   └── ui/                # Reusable UI components
├── sections/              # Page sections (Hero, Contact, etc.)
├── data/                  # Site configuration and content
└── lib/                   # Utilities and helpers

public/
├── images/                # Static images
├── robots.txt             # Search engine configuration
├── sitemap.xml            # XML sitemap
└── llms.txt               # LLM discoverability file
```
